# 🧩 Media Tracker - Chrome Extension

Bu klasör, popüler yayın platformlarından (Netflix, Prime Video, vb.) izleme verilerini otomatik olarak yakalayan Chrome uzantısını içerir.

## 🚀 Kurulum

1.  Chrome tarayıcınızda `chrome://extensions/` adresine gidin.
2.  Sağ üstteki **Geliştirici Modu** (Developer Mode) anahtarını açın.
3.  **Paketlenmemiş öğe yükle** (Load unpacked) butonuna tıklayın.
4.  Bu klasörü (`media-extension`) seçerek yükleyin.

## 🛠️ Özellikler

-   İzlenen içeriğin adını ve sezon/bölüm bilgisini otomatik algılama.
-   Arka planda Backend API ile senkronizasyon.
-   Basit ve hafif yapı.
