# 🎨 Media Tracker - Web Frontend

Bu klasör, platformun React + Vite tabanlı modern kullanıcı arayüzünü içerir.

## 🚀 Başlangıç

1.  Bağımlılıkları yükleyin:
    ```bash
    npm install
    ```
2.  Geliştirme sunucusunu başlatın:
    ```bash
    npm run dev
    ```

## 📂 Teknolojiler

-   **React 18**
-   **Vite**
-   **TailwindCSS** (veya Vanilla CSS)
-   **Axios** (API İletişimi)
-   **Lucide React** (İkonlar)
